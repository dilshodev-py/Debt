HTML Select Tag Examples

1. Basic Select Dropdown

<select id="fruits" name="fruits"> <option value="">--Please choose an option--</option> <option value="apple">Apple</option> <option value="banana">Banana</option> <option value="orange">Orange</option> </select>

2. Select with Preselected Option

<select id="countries" name="countries"> <option value="us">United States</option> <option value="uk" selected>United Kingdom</option> <option value="ca">Canada</option> </select>

3. Multiple Selection

<select id="colors" name="colors" multiple> <option value="red">Red</option> <option value="blue">Blue</option> <option value="green">Green</option> </select>

4. Grouped Options (Optgroup)

<select id="vehicles" name="vehicles"> <optgroup label="Cars"> <option value="sedan">Sedan</option> <option value="suv">SUV</option> </optgroup> <optgroup label="Motorcycles"> <option value="sport">Sport Bike</option> </optgroup> </select>

5. Styled Select

<select class="styled-select"> <option value="low">Low Priority</option> <option value="medium">Medium Priority</option> <option value="high">High Priority</option> </select>

6. Large Select with Size Attribute

<select id="numbers" name="numbers" size="4"> <option value="1">One</option> <option value="2">Two</option> <option value="3">Three</option> </select>

7. Interactive Select with JavaScript

<select onchange="showSelection()"> <option value="html">HTML</option> <option value="css">CSS</option> </select> <script> function showSelection() { const select = document.getElementById('interactive'); const result = document.getElementById('selectionResult'); if (select.value) { result.textContent = `You selected: ${select.value.toUpperCase()}`; result.style.display = 'block'; } else { result.style.display = 'none'; } } </script>
